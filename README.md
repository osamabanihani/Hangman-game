# 🎮 Hangman Game (React)

A modern and interactive **Hangman Game** built with React.
Guess the hidden word before the full drawing appears!

---

## 🚀 Features

* 🎯 Random word generation
* ⌨️ Interactive on-screen keyboard
* 🎨 Dynamic hangman drawing (step-by-step)
* 🧠 Win / Lose logic detection
* 🔁 Restart game functionality
* 📱 Responsive design

## 🎮 How It Works

* A random word is selected when the game starts
* You guess letters using the keyboard
* Each wrong guess adds a new body part
* Maximum wrong guesses: **6**
* You win when you guess the full word
* You lose when the drawing is complete

```bash id="cmd22"

```

Install dependencies:


